# Data panel

Reserved for future data tools (e.g. re-sync from the Geometry Dashboard,
parameter mapping, round-trip export).

Note: Revit does not render ribbon panels that contain no buttons, so this
panel stays invisible until the first `*.pushbutton` folder is added here.
Its slot is already first in `../bundle.yaml`, so when populated it will
appear on the far left of the pyGherkin tab.
